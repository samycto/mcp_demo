Getting started
Three steps to your first Enterprise MCP connection
1
Access the MCP registry in your Workato workspace under AI Hub → MCP Servers. Browse the prebuilt servers for your selected systems. View Prebuilt Servers →
2
Connect your first server. Pick the system with the highest daily use — Salesforce, Slack, or your email client are strong starting points. Authenticate and activate. View MCP server setup docs →
3
Connect your LLM to the Workato MCP Gateway. Copy your MCP endpoint URL, paste it into your LLM's MCP configuration, and run your first prompt. View LLM configuration docs →
